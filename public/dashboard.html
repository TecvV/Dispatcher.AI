Trusted Contacts

Add doctor, psychiatrist, family, friends, partner, or custom contact type. These can be used for crisis outreach and meet scheduling.

How to get Telegram Chat ID:
1. Ask contact to open Telegram and search @RawDataBot.
2. Ask them to send /start. They will get JSON; use message -> chat -> id.
3. Ask them to share that ID and enter it above.

Discord Channels (Independent)

Add Discord webhook channels independently. Chat will ask which channel to use when multiple channels exist.

How to get Discord Channel Webhook:
1. Open your Discord server channel settings for the target text channel.
2. Go to Integrations -> Webhooks and click Create Webhook.
3. Choose the channel, then click Copy Webhook URL and paste it above.
